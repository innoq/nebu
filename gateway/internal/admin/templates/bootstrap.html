Step 3: Key Generation

Cryptographic Key Pairs

Nebu will generate two key pairs for your instance:

  • Ed25519 — signing key pair for event signatures and identity verification
  • X25519 — encryption key pair for end-to-end encrypted message content

Keys are generated server-side and stored securely. The public Ed25519 fingerprint will be shown for verification.
