## Key Discoveries

1. **Dormant-defect activation pattern.** `media/internal/upload/upload.go` has had `// for MVP, accept any bearer` since Story 4-19. Epic 12 deployed the media service on port 8009 — the latent auth gap became a reachable HIGH. Generalised into MEMORY.md as a recurring pattern: when a code-only module is promoted to a deployed compose service, every `// MVP` / `// placeholder` token in the affected files must be re-audited.

2. **createbuckets entrypoint defeats Docker Secrets.** `$(cat /run/secrets/...)` inside `mc admin user add minio $(...) $(...)` puts the secret on `mc` argv → `/proc/<pid>/cmdline`. The MinIO recommended fix is `MC_HOST_<alias>` env or `--stdin`. Added as recurring pattern.

3. **`USING (true) WITH CHECK (true)` on a key-value table.** Migration 000045 silently reverted the ADR G8 immutability guarantee for `server_config`. The right fix is a key-scoped `USING (key IN (...))`. Added as recurring pattern.

4. **Unauthenticated + unbounded resource on Matrix v3 endpoints.** The thumbnail endpoint validates `width > 0` / `height > 0` but has no upper bound — `imaging.Fit` allocates `4 * width * height` bytes per request. Combined with no rate-limit middleware on the media gateway, this is a single-request OOM.

## What Worked

- Carrying the recurring-pattern table from prior epics surfaced the RLS regression on `server_config` immediately by triage (same shape as Epic 9 RLS-omission pattern).
- Quick win: dev/minio/nebu-app-policy.json is **correctly** least-privilege (no DeleteObject, no `s3:*`, no ListBucket; scoped to `nebu-media/*`) — the test file `minio_policy_test.go` even enforces this. Story 12.3 IAM hardening was done right.
- gRPC trust-boundary pattern (GetEvent uses `Nebu.Grpc.Metadata.trusted_identity`) continues to be applied correctly — Epic 11 lesson stuck.

## What Could Be Improved

- ATDD step did not produce a "DoS / decompression bomb" acceptance test for the thumbnail handler. The Acceptance Tests section in Story 12.5 should have explicitly enumerated "reject width > N" and "reject decoded source > N megapixels". Worth feeding back to TEA.
- The diff range bundled Epic 11 carry-over stories with Epic 12 — separating those at branch level would make epic-end reviews cleaner.
