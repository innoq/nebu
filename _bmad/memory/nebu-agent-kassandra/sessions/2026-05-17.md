### Lessons / Lasting Patterns

- **Proto field reservation = strongest fix for caller-controlled string fields.** Better than runtime allowlist because it removes the wire-level marshal capability. Use this shape whenever a `system_role` / `is_admin` / `role` field leaks into a request payload that should be server-decided.
- **Realistic-shape test pinning.** AT-7 deliberately mismatches `ContextKeySub` and `ContextKeyUserID` so the guard *must* use the correct one to pass. This is the test pattern that catches the "F-1 family" of identity-confusion bugs at red-phase rather than at SEC Gate 2.
- **Follow-up story consolidation.** Original review demanded Stories 14.5 + 14.6 — Phil chose a single security-fix commit instead. Acceptable when (a) regression tests pin every claimed fix, (b) re-review verifies, (c) the commit message names the finding IDs. The audit narrative is: commit `1e01563` + this re-review.
