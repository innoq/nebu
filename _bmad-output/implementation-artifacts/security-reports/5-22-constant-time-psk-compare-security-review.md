## Detailed Analysis

### 1. Hash-then-Compare Correctness

```go
func hashForCompare(value []byte) []byte {
    mac := hmac.New(sha256.New, []byte(pskDomain))
    mac.Write(value)
    return mac.Sum(nil)
}
```

- Uses `hmac.New` (not raw `sha256.Sum256`), providing domain separation and resistance to length-extension attacks.
- Both `authHeader` and `expected` are hashed independently before comparison.
- Output is always exactly 32 bytes, making `subtle.ConstantTimeCompare` truly constant-time.

### 2. Domain Separator

- Value: `"nebu/psk/v1"` -- compile-time `const`, not derived from user input.
- Prevents an attacker who knows the HMAC-SHA256 of some other value from reusing that digest here.
- Versioned (`v1`) to allow rotation if the scheme changes.

### 3. No Secret Leakage in Logs

- `psk.go` contains zero logging statements.
- `PSKMiddleware` returns bare `401` with empty body on mismatch -- no error detail, no header echo.

### 4. CSRF Comparison Audit

- `admin/middleware.go:281`: `subtle.ConstantTimeCompare([]byte(cookieVal), []byte(formVal))`.
- Both values are base64url-encoded 32-byte randoms (43 chars fixed). Same-length guarantee holds.
- No hash-then-compare needed. Documented as OK.
